Purpose

Ensure all Right of Entry forms are accurately received, verified, processed, recorded, and communicated across all internal systems.

Bloomerang (CRM) Google Drive Google Sheets Google MyMaps Adobe Acrobat documents@theclerc.org

Step 1 — Identify the ROE Submission Type

✉
Physical (Mailed Hard Copy) Follow Path A below
💻
Digital (Email Approval) Skip to Path B below

Path A Physical (Mailed) ROEs

A1

Mail Intake & Sorting

  1. Open all incoming ROE mail.
  2. Separate ROEs into piles by grant (typically CWDG-1 or CWDG-2).
  3. Keep ROEs grouped by grant to maintain reporting accuracy and minimize context-switching.
💡
How to determine grant: Recognize the project area/address, or search the property owner/address in Bloomerang and check the "Groups" field. If unsure, set aside and resolve before scanning.
⏲
Time tracking: Charge sorting time to Admin. Once working within a specific grant, switch time to that grant.
A2

Scanning Documents

  1. Scan all ROEs using the office scanner.
  2. Each ROE must be its own individual PDF (do not combine).
  3. Scan in the same order as your sorted piles.
  4. Save to: Google Drive → My Drive → From Brother Device
⚠
Confirm all pages are legible and nothing is cut off or rotated incorrectly.
A3

Begin Processing (By Grant)

  1. Open the scanned folder.
  2. Select the first ROE from one grant pile.
  3. Switch time tracking to that grant.
A4

Identify Property Owner & Project

  1. Open the scanned ROE and locate the property owner name and property address.
  2. Search in Bloomerang — first by name, then by address if not found.
  3. If still not found, search the project progress spreadsheet by address.
  4. Open the correct profile and confirm the project under Profile → Groups (e.g., "Bottle Rock Roadside 2026").
A5

Open Required Project Files

Open all four of the following in separate tabs:

  1. Project Google Drive folder
  2. ROE folder (within the project folder)
  3. Project progress tracker (Google Sheet)
  4. Project Google MyMaps
A6

Accept & Complete ROE (Adobe Acrobat)

  1. Open the ROE in Adobe Acrobat.
  2. Use Fill & Sign to complete: Signature, Title, Date, Grant Code.
  3. Close the Fill & Sign tool.
  4. Save As → Choose a Different Folder → navigate to the correct project ROE folder.

File naming convention:

[Property Owner Name] – ROE – [Project Name] – [MM.DD.YY]
A7

Update Bloomerang Record

Contact Information — Add phone, email, or alternate mailing address if provided on the ROE.

⚠
If different info already exists, add the new info as primary. Do NOT delete existing data.
If owned by a business or trust

Click "Edit" and add the primary contact person's name.

ROE Status — Scroll to "Landowner Information" and set Current ROE = YES. Save.

A8

Add Bloomerang Note

  1. Click Add Note.
  2. Enter: ROE signed by [Property Owner Name]. Accepted and Returned by [Your Name]
  3. Click + → Web Link → paste the Google Drive file link.
  4. Save the note.
A9

Update Project Progress Tracker

  1. Locate the property row(s) in the tracker.
  2. Highlight the full row(s) with light green fill.
  3. In the Notes field, insert a hyperlink labeled "2026 ROE" linking to the ROE file in Drive.
  4. Fill in: primary contact name, phone number, email.
A10

Update Google MyMaps

For each address listed on the ROE:

  1. Search the address in the map.
  2. Click the existing red marker → delete it (trash icon).
  3. Select the "ROE Received" layer.
  4. Re-search the address and add it to the map under this layer.
  5. Confirm the marker now appears as a green check.
A11

Send Confirmation

If email was provided
  1. Open documents@theclerc.org → Compose.
  2. Open Templates → "Manual: Thank You - Right Of Entry Form Received"
  3. Replace [First Name] and remove "Manual:" from the subject line.
  4. Attach the accepted ROE.
  5. Send.
If no email was provided
  1. Print the accepted ROE.
  2. Open Google Docs → Template Gallery → "ROE Physical Acceptance Letter"
  3. Personalize the letter and print.
  4. Prepare envelope: print mailing label, add postage.
  5. Mail the acceptance letter + copy of ROE.
A12

File Physical Copy & Final Update

  1. Mark the ROE as completed (highlight/mark).
  2. File in the physical filing cabinet.
  3. Return to the project progress sheet and check the "Returned" checkbox.

Path B Digital ROEs

B1

Intake & Review

  1. Open documents@theclerc.org and locate the ROE approval email.
  2. Click "Review & Approve".
  3. Scroll to confirm the property owner name and property address.
B2

Identify Property & Open Project Files

Follow the same lookup process as the physical path:

  1. Search in Bloomerang by name, then address. Same as A4
  2. Confirm the project via Profile → Groups.
  3. Switch time tracking to the correct grant.
  4. Open all four project files in separate tabs. Same as A5
B3

Accept & Save ROE

  1. Complete the digital approval.
  2. On the confirmation screen, click "Download a Copy".
  3. Rename the file using the standard naming convention:
[Property Owner Name] – ROE – [Project Name] – [MM.DD.YY]
  1. Upload to the project's ROE folder in Google Drive.
B4

Update All Systems

Complete the same system updates as the physical path:

  • Update Bloomerang contact info & ROE status A7
  • Add Bloomerang note with Drive link A8
  • Update project progress tracker A9
  • Update Google MyMaps markers A10
B5

Send Confirmation & Final Update

  1. Open documents@theclerc.org.
  2. Use template: "Thank You - Right Of Entry Form Received"
  3. Replace [First Name] and send.
  4. Return to the tracker and check the "Returned" box.

✅ Final Completion Checklist

Confirm all items before closing any ROE.
